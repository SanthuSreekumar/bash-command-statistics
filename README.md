# Bash Command Statistics Analyzer (Linux + Python)

A Linux based Python tool that analyzes bash history and generates command usage statistics. The tool helps understand command usage patterns by identifying frequently used commands and allowing quick search and report generation.

## Features
• Top 10 most used commands analysis 
• Command usage search functionality 
• Report generation to text file
• Menu driven interface
• Basic data cleaning for invalid entries

## Tech Stack
Python, Linux, Bash History, Collections module

## Purpose
Built to practice Python file handling, data processing, and basic system analysis.
